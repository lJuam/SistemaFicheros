# SistemaFicheros
# Listado y Caracterización de Ficheros en Directorios - Sistemas Operativos

Este repositorio contiene dos programas desarrollados en lenguaje C para la asignatura Sistemas Operativos de la Pontificia Universidad Javeriana. Los programas permiten el análisis y listado de ficheros en un directorio del sistema de archivos Linux/Unix.

- `ficherosLab01.c`: Programa que lista los ficheros regulares de un directorio ingresado por teclado e imprime su nombre y tamaño en bytes.
- `ficherosLab02.c`: Programa que muestra información detallada de los ficheros y directorios contenidos en un directorio, incluyendo permisos, modo y fecha de modificación.

Ambos programas se compilan y manejan mediante el Makefile incluido, que automatiza la compilación y limpieza de los ejecutables.

## Uso

1. Clonar el repositorio.
2. Ejecutar `make` para compilar ambos programas.
3. Ejecutar `./ficherosLab01` o `./ficherosLab02` y seguir las indicaciones en consola para ingresar rutas de directorios.

## Documentación

Los archivos `.c` cuentan con documentación detallada, comentarios explicativos y observaciones sobre funcionamiento y uso de llamadas al sistema.
